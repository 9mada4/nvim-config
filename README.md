# Neovim setup

## 1. Install Neovim
```zsh
brew install nvim
```

## 2. Clone config
1.
```zsh
git clone git@github.com:9mada4/nvim-config.git ~/.config/nvim
```
or
1. unzip nvim-config-main.zip 
```zsh
cd ~/Downloads
mkdir -p ~/.config
mv nvim-config-main ~/.config/nvim
```
2. `ls ~/.config` -> nvim ok

## 3. Open Neovim
nvim
